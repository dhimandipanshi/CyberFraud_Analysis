# 🚨 Cyber Fraud Analysis in Canada (2021–2025)

## 📌 Overview
This project analyzes **cyber fraud trends across Canada** using publicly available data from the **Canadian Anti-Fraud Centre (CAFC)**.  
The analysis focuses on identifying **fraud patterns, financial impact, and high-risk demographics**, supported by an **interactive Power BI dashboard** designed for stakeholder insights.

---

## 🎯 Business Objective
- Understand the scale and financial impact of cyber fraud in Canada  
- Identify the most prevalent fraud types and affected regions  
- Analyze victim demographics to highlight high-risk groups  
- Support data-driven decision-making for fraud prevention and awareness  

---

## 📂 Dataset
- Source: **Canadian Anti-Fraud Centre (CAFC)**  
- Time period: **2021–2025**  
- Scope: Nationwide cyber fraud incidents and reported losses  

**Key fields include:**
- Fraud type  
- Province and region  
- Victim age group  
- Number of victims  
- Financial loss amount  

---

## 🧹 Data Preparation (Excel)
Initial data preparation was performed in **Microsoft Excel**, including:

- Removal of duplicate and incomplete records  
- Standardization of fraud categories and provinces  
- Formatting financial values for accurate aggregation  
- Validation of year-over-year trends  

📂 **Cleaned Dataset:** `data/cyberfraud_data.xlsx`

---

## 📊 Key Findings
- **295,000+ victims** reported cyber fraud incidents between 2021–2025  
- **Total losses exceeded $637 million**, indicating significant economic impact  
- **Investment scams** accounted for the highest financial losses  
- **Ontario and Quebec** reported the largest number of fraud cases  
- Individuals aged **30–59** were the most commonly affected demographic  

---

## 📈 Power BI Dashboard
An interactive **Power BI dashboard** was built to visualize fraud trends and enable exploratory analysis.

**Dashboard Features:**
- Fraud incidents and losses over time  
- Breakdown by fraud type  
- Geographic distribution across provinces  
- Victim age-group analysis  

📂 **Dashboard File:** `dashboard/CyberFraudDashboard.pbix`  
📸 **Preview:** `images/dashboard_preview.png`

---

## 💡 Insights & Implications
- Financial impact is highly concentrated in a few fraud categories  
- High-population provinces show both higher volume and losses  
- Mid-age working professionals are a key risk group  
- Preventive efforts should prioritize investment-related scams  

---

## 🚀 Recommendations
- Strengthen public awareness campaigns around investment scams  
- Target fraud-prevention messaging toward high-risk age groups  
- Use geographic insights to prioritize regional enforcement efforts  
- Monitor year-over-year trends to evaluate policy effectiveness
